魔法動作 #

施展咒語 #

當你嘗試施展一個你已經學會的咒語時，先念出咒語並揮舞魔杖，接著擲+魔法。
如果這是你不會的咒語，不要念出咒語或者揮舞魔杖，你必須標記一點幸運，然後擲+魔法。

在沒有失敗風險的情況下，施展咒語並不需要擲骰。通常來說，除非你處在一個有壓力的情境下，否則你永遠能夠成功施展等級低於你目前年級兩年以下的咒語。

擲出10+時，你成功依照你所想的方式施展咒語。

擲出7-9時，你成功施展咒語，但是……（選擇一項）

  • 咒語的效果沒有發揮它本來應有的強度。
  • 咒語的效果沒有維持它本來應有的持續時間。
  • 你吸引了不必要的注意。

擲出6-時，標記一點經驗值。你沒能成功施展咒語，且敘事者還可以告訴你一個因此發生的嚴重後果。

調製魔藥 #

當你攪拌、熬煮、或者用其他的方式配製一劑你已經學會的魔藥時，擲+魔法。
你必須有正確的材料，掌握組合材料的方式，以及一根魔杖。當你嘗試配製你不會的魔藥時，你必須標記一點幸運，接著擲+魔法。

擲出10+時，你正確地調製出你想要的魔藥。

擲出7-9時，你做出了魔藥，但是……（選擇一項）

  • 它有一些意料之外的副作用。
  • 製作過程出了點亂子，你標記一個狀態。
  • 你不小心做出了另一種魔藥。（敘事者會告訴你做出了什麼）

擲出6-時，標記一點經驗值。你沒能成功做出魔藥，且敘事者還可以告訴你一個因此發生的嚴重後果。

使用魔法道具 #

當你使用魔法道具時，擲+魔法。

擲出10+時，你充分利用了道具並讓它發揮預期中的作用。

擲出7-9時，選擇一項：

  • 道具造成了意料之外但有幫助的效果。
  • 道具的效果並沒有發揮它本來應有的強度。
  • 道具發揮了預期中的作用，但是你因為使用它的過程而標記一個狀態。
  • 道具發揮了預期中的作用，但是在過程中壞掉了。

擲出6-時，標記一點經驗值。魔法道具沒有正確發揮效果，且敘事者還可以告訴你一個因此發生的嚴重後果。

應付奇獸 #

當你嘗試馴服、幫助奇獸、尋求奇獸的幫助、或透過其他方式與奇獸互動時，擲+忠誠。

擲出10+時，奇獸照你所想的方式反應。

擲出7-9時，選擇一項：

  • 奇獸照你所想的方式反應，但你在過程中標記了一個狀態。
  • 奇獸照你所想的方式反應，但牠同時也吸引了不必要的注意。
  • 奇獸沒有照你所想的方式反應，但是牠做出的舉動仍然對你有幫助。

擲出6-時，標記一點經驗值。奇獸發狂，且敘事者還可以告訴你一個因此發生的嚴重後果。

決鬥 #

當你捲入一場巫師決鬥時，擲+魔法來施展咒語，但是使用以下的選項：

擲出10+時，你的咒語擊中了對手女巫 / 巫師，且他們的咒語沒有擊中你！

擲出7-9時，選擇一項：

  • 你們的咒語在空中對撞了！
  • 你的咒語沒有命中，但是他們的咒語也沒有！
  • 雙方的咒語都命中了！

擲出6-時，標記一點經驗值。你的咒語沒能命中，但對手的咒語命中了你！敘事者將告訴你一個因此發生的嚴重後果。

如果你施展的是防禦或治療咒語的話，把上述效果的「命中」視為格擋或成功解消效果。決鬥會在其中一方的女巫 / 巫師投降或再也無法施展咒語時結束。


如果你在與另一名主要角色決鬥的話……

首先，雙方擲+英勇。由擲骰結果比較高的女巫 / 巫師先開始他的回合。

當輪到你的回合時，雙方的女巫 / 巫師揮舞魔杖並說出他們打算施展的咒語。接著，擲+魔法（另一位女巫 / 巫師不擲骰）。使用上述選項來決定這回合決鬥的結果。接著輪到另一位女巫 / 巫師的回合。
